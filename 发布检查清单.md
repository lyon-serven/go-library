# MyLib 发布前检查清单

在发布到仓库之前，请确保完成以下所有步骤：

## 📋 发布前准备

### ✅ 1. 仓库创建

- [ ] 已在 GitHub 或 Gitee 创建仓库
- [ ] 仓库设置为 **公开** (Public)
- [ ] 记录仓库完整路径（例如: `github.com/yourname/mylib`）

### ✅ 2. 模块配置

- [ ] 运行了配置脚本或手动更新了 `go.mod`
- [ ] `go.mod` 第一行是完整的仓库路径
  ```bash
  head -n 1 go.mod
  # 应显示: module github.com/yourname/mylib
  ```

### ✅ 3. 导入路径更新

- [ ] 所有 `.go` 文件的导入路径已更新
- [ ] 检查关键文件的导入:
  ```bash
  grep "import.*mylib" integration_demo.go examples/*.go util/*/examples/*.go
  # 不应该有 "mylib/" 开头的导入，应该是完整路径
  ```

### ✅ 4. 编译测试

- [ ] 运行 `go mod tidy` 成功
  ```bash
  go mod tidy
  ```
  
- [ ] 编译无错误
  ```bash
  go build ./...
  ```
  
- [ ] 集成测试通过
  ```bash
  go run integration_demo.go
  ```

### ✅ 5. 文件检查

- [ ] `.gitignore` 文件已创建
- [ ] 移除了不必要的临时文件
- [ ] 确认以下文件存在且完整:
  - [ ] `README.md` - 项目介绍
  - [ ] `go.mod` - 模块配置
  - [ ] `LICENSE` - 许可证（可选但建议添加）

### ✅ 6. Git 配置

- [ ] Git 已初始化
  ```bash
  git status
  ```
  
- [ ] 所有文件已添加
  ```bash
  git add .
  git status  # 确认没有遗漏文件
  ```
  
- [ ] 已创建初始提交
  ```bash
  git commit -m "feat: initial commit"
  ```
  
- [ ] 已连接远程仓库
  ```bash
  git remote -v
  # 应显示 origin 和仓库地址
  ```

### ✅ 7. 推送代码

- [ ] 代码已推送到远程仓库
  ```bash
  git push -u origin main  # 或 master
  ```
  
- [ ] 已创建版本标签
  ```bash
  git tag v1.0.0
  git push origin v1.0.0
  ```
  
- [ ] 在 GitHub/Gitee 网页上确认代码可见

### ✅ 8. 验证可用性

- [ ] 在新目录测试下载
  ```bash
  cd ../
  mkdir test-download
  cd test-download
  go mod init test
  go get github.com/yourname/mylib@v1.0.0
  ```
  
- [ ] 创建并运行测试代码


## 🧪 验证代码

创建 `test.go` 文件测试各个模块：

```go
package main

import (
    "fmt"
    "github.com/yourname/mylib/util/authutil"
    "github.com/yourname/mylib/util/cryptoutil"
    "github.com/yourname/mylib/util/timeutil"
)

func main() {
    fmt.Println("=== MyLib 功能测试 ===\n")
    
    // 测试 Google Authenticator
    fmt.Println("1. Google Authenticator:")
    secret, qrURL, err := authutil.QuickGenerate("TestApp", "test@example.com")
    if err != nil {
        fmt.Printf("❌ 错误: %v\n", err)
    } else {
        fmt.Printf("✅ 密钥: %s\n", secret[:20]+"...")
        fmt.Printf("✅ QR码: %s\n", qrURL[:50]+"...")
    }
    
    // 测试加密工具
    fmt.Println("\n2. 加密工具:")
    hash := cryptoutil.SHA256("test")
    fmt.Printf("✅ SHA256: %s\n", hash[:16]+"...")
    
    // 测试时间工具
    fmt.Println("\n3. 时间工具:")
    now := timeutil.Now()
    fmt.Printf("✅ 当前时间: %s\n", now.Format("2006-01-02 15:04:05"))
    
    fmt.Println("\n🎉 所有功能正常！")
}
```

运行测试:
```bash
go run test.go
```

## 🔍 常见问题自查

### 问题1: go get 失败

**检查项:**
- [ ] 仓库是公开的吗？
- [ ] 代码和标签都推送了吗？
  ```bash
  git ls-remote --tags origin
  # 应该看到 v1.0.0
  ```
- [ ] go.mod 中的 module 路径正确吗？
- [ ] 网络连接正常吗？

### 问题2: 导入路径错误

**检查项:**
- [ ] 所有文件都更新了吗？
  ```bash
  grep -r "\"mylib/" . --include="*.go"
  # 应该没有结果
  ```
- [ ] 运行了 `go mod tidy` 吗？
- [ ] 重新推送了代码吗？

### 问题3: 编译错误

**检查项:**
- [ ] 本地编译成功吗？
  ```bash
  go build ./...
  ```
- [ ] 依赖都正确吗？
  ```bash
  go mod verify
  ```
- [ ] 清理缓存重试？
  ```bash
  go clean -modcache
  go mod tidy
  ```
