# JWT工具类改进清单

## ✅ 已完成的改进

### 1. 修复HS384和HS512算法实现 ✅
- **问题**: HS384使用了错误的sha256.New224,HS512重复使用sha256.New
- **修复**: 
  - HS384改为使用 `sha512.New384`
  - HS512改为使用 `sha512.New`
- **状态**: 已修复

### 2. 添加密钥强度验证 ✅
- **新增**: `MinSecretKeyLength` 常量(32字节)
- **新增**: `validateSecretKey()` 方法
- **功能**: 确保密钥长度至少为256位,提高安全性
- **状态**: 已实现

### 3. 添加时钟偏移容忍 ✅
- **新增**: `ClockSkew` 配置字段
- **新增**: `DefaultClockSkew` 常量(5秒)
- **功能**: 在验证exp和nbf时允许5秒的时间偏差
- **状态**: 已实现

### 4. 改进安全警告 ✅
- **改进**: `ParseTokenWithoutVerify` 函数注释
- **新增**: 多行警告说明不安全用途
- **状态**: 已完成

### 优先级 P3 (低优先级)

#### 11. 添加Context支持
```go
import "context"

func (c *JWTConfig) GenerateTokenWithContext(ctx context.Context, claims *Claims) (string, error) {
    select {
    case <-ctx.Done():
        return "", ctx.Err()
    default:
        return c.GenerateToken(claims)
    }
}

func (c *JWTConfig) VerifyTokenWithContext(ctx context.Context, tokenString string) (*Claims, error) {
    select {
    case <-ctx.Done():
        return nil, ctx.Err()
    default:
        return c.VerifyToken(tokenString)
    }
}
```
