# Google Authenticator 功能更新 - 国内可用版本

## 📢 更新说明

针对 Google Charts API 在国内无法访问的问题，现已添加多种国内可用的二维码生成方案。

## ✨ 新增功能

### 1. 新增国内可访问的API方法

```go
// 使用国内可访问的 QR Server API
qrURL := ga.GenerateQRCodeImageURLCN(secret)
// 返回: https://api.qrserver.com/v1/create-qr-code/?size=200x200&data=...
```

### 2. 新增获取原始URL方法

```go
// 获取原始 otpauth:// URL，可用于任何二维码生成工具
otpauthURL := ga.GetOtpauthURL(secret)
// 返回: otpauth://totp/MyApp:user@example.com?secret=...
```

### 3. 更新快速API默认使用国内可访问的服务

```go
// QuickGenerate 现在默认使用国内可访问的 QR Server API
secret, qrURL, _ := authutil.QuickGenerate("MyApp", "user@example.com")

// 如果需要使用 Google Charts API（需要翻墙），使用新的方法
secret, qrURL, _ := authutil.QuickGenerateWithGoogle("MyApp", "user@example.com")
```

### 4. 新增本地生成二维码的辅助文件

- `qrcode_local.go` - 本地生成二维码的示例和说明
- 支持使用 `github.com/skip2/go-qrcode` 库本地生成

### 5. 新增完整示例文件

- `examples/qrcode_solutions.go` - 展示所有二维码生成方案的完整示例

## 🚀 快速使用

### 场景1: Web应用 - 本地生成（推荐）

```go
// 1. 安装依赖
// go get github.com/skip2/go-qrcode

import "github.com/skip2/go-qrcode"

// 2. 生成密钥和QR码
config := authutil.DefaultTOTPConfig()
config.Issuer = "MyApp"
config.AccountName = userEmail
ga := authutil.NewGoogleAuthenticator(config)

secret, _ := ga.GenerateSecret()
otpauthURL := ga.GetOtpauthURL(secret)

// 3. 生成base64编码的QR码图片
png, _ := qrcode.Encode(otpauthURL, qrcode.Medium, 256)
base64Str := base64.StdEncoding.EncodeToString(png)

// 4. 返回给前端
response := map[string]string{
    "secret": secret,
    "qrDataUrl": "data:image/png;base64," + base64Str,
}
```

### 场景2: 快速开发 - 在线API

```go
// 使用内置的快速API（现已默认使用国内可访问的服务）
secret, qrURL, err := authutil.QuickGenerate("MyApp", "user@example.com")

// qrURL 可直接在浏览器打开或在 <img> 标签中使用
// <img src="{{qrURL}}" alt="扫描二维码" />
```

### 场景3: 命令行工具 - 终端显示

```go
// 1. 安装依赖
// go get github.com/mdp/qrterminal/v3

import qrterminal "github.com/mdp/qrterminal/v3"

// 2. 生成并显示
config := authutil.DefaultTOTPConfig()
config.Issuer = "MyApp"
config.AccountName = "user@example.com"
ga := authutil.NewGoogleAuthenticator(config)

secret, _ := ga.GenerateSecret()
otpauthURL := ga.GetOtpauthURL(secret)

// 在终端显示二维码
qrterminal.Generate(otpauthURL, qrterminal.L, os.Stdout)
fmt.Printf("密钥: %s\n", secret)
```
